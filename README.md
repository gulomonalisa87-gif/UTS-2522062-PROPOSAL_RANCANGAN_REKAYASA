# UTS-2522062-PROPOSAL_RANCANGAN_REKAYASA
Sistem kendali mutu otomatis berbasis Edge Intelligence pada lini produksi manufaktur menggunakan ESP32. Tugas  UTS Pengantar Rekayasa Dan Desain ITEBA
## File Kode
-'kode-sensor.ino': Program baca sensor suhu dan berat
-'Kode-kontrol.ino': Program kontrol motor servo dan relay

## Cara Pakai
1. Buka'kode-sensor.ino' di Arduino IDE, upload ke ESP32
2. Buka'Kode-kontrol.ino', upload ke ESP32
